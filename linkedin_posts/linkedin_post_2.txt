MATCH is like SQL INNER JOIN. ALL relationships must exist.

Devil's Advocate has director but no producer. MATCH fails.

Fix? OPTIONAL MATCH (LEFT JOIN):

MATCH (keanu:Person {name: 'Keanu Reeves'})-[:ACTED_IN]->(movie)
OPTIONAL MATCH (movie)<-[:DIRECTED]-(director)
OPTIONAL MATCH (movie)<-[:PRODUCED]-(producer)
RETURN movie.title, COLLECT(DISTINCT director.name), COLLECT(DISTINCT producer.name)
